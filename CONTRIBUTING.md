# Contributing

Thanks for your interest in improving JVM Diagnostics Analyzer.

## How to contribute

1. Fork the repository and create a branch for your change.
2. Keep changes focused and include/update tests when relevant.
3. Run `mvn test` before opening a pull request (Java 25 is required).
4. Open a pull request with a clear description of the problem and solution.

## Code of Conduct

Please be respectful and constructive in all interactions. By participating in this project, you agree to collaborate professionally, welcome feedback, and avoid harassment or discriminatory behavior.
